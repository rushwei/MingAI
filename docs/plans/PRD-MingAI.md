## 1. 产品概述

### 1.1 产品愿景

打造一个**多端可用、可解释、可记忆**的AI命理服务平台，将传统命理文化与前沿AI技术深度融合，为用户提供专业、私密、便捷、随时随地的命理咨询服务。

### 1.2 产品定位

| 维度           | 描述                                                         |
| -------------- | ------------------------------------------------------------ |
| **目标市场**   | 中国大陆及海外华人市场                                       |
| **产品形态**   | 响应式Web应用 + iOS/Android 客户端（Capacitor）              |
| **核心价值**   | AI驱动的个性化命理分析 + 知识库长期记忆 + 多端随时可用        |
| **差异化优势** | 流式对话、@提及/来源可解释、统一数据源、隐私优先、可运营体系 |

### 1.4 目标用户画像

| 用户类型         | 特征                             | 核心需求                         |
| ---------------- | -------------------------------- | -------------------------------- |

### Phase 8.6 MCP服务 (已实现)

| RAG      | Retrieval-Augmented Generation，检索增强生成              |
| @提及    | 在对话中显式引用知识库或数据源内容                         |
| pgvector | PostgreSQL 向量检索扩展                                    |
| 推理模式 | 高成本/高精度AI推理能力（Reasoning）                       |
